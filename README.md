# Desenvolvimento Web - Lojinha de Poções
Projeto 2 da disciplina de Desenvolvimento Web.

## Execução
Para executar o código, em um terminal, execute o comando enquanto na pasta front:
`npx http-server --cors -c-1`

Em seguida, na pasta back, execute:
`node index.js`

Por fim, abra o projeto no localhost 8080. Para acessar a página de administração, acesse o `/admin.html`
